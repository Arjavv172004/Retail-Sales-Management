# Frontend Application

## Overview

Modern React application built with Vite for the Retail Sales Management System. Features a beautiful, animated UI with smooth transitions and responsive design.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Features

- **Search**: Real-time search across Customer Name and Phone Number
- **Filters**: Multi-select filters for Region, Gender, Category, Tags, Payment Method
- **Range Filters**: Age and Date range filters
- **Sorting**: Sort by Date, Quantity, or Customer Name
- **Pagination**: Navigate through results with 10 items per page
- **Animations**: Smooth transitions and hover effects using Framer Motion
- **Responsive**: Mobile-friendly design

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- React Query
- Axios
